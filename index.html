Projects that I have Worked On
Axel Wilson's Mixtape Maker

Axel Wilson's Mixtape Maker was developed during Cornhacks 2022 by AJ Richerson, Elijah Smith, Rylen Lanning, and myself. Becasause the theme of the hackathon was 'retro', we settled on the aesthetic of 'Bill and Ted's Excellent Adventure' with the our fake person Axel Wilson who 'owns' the site.

The web app leverages the recommendation system from Spotify Web API through Spotipy to get song recommendations on a single-song basis. After choosing a base song to start with, the API will reccomend five songs from which the user can choose the songs they would like to proceed with. Songs chosen will be passed through the recommendation system to generate new songs. Songs will only be reccomended based off of the songs chosen on in the previous set, where no songs will be recommended twice. At the end comes the 'Mixtape' part, where all chosen songs will be compiled into a playlist.

My contribution to the project consisted of planning group meetings beforehand to get the group members aquainted with one another, setting out general project ideas, developing the frontend of the web app using CSS and HTML and collaborating with AJ who was familiar with React, and sifting through some troubleshooting errors with some users on Mac.
